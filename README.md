# Homebrew Tap

Homebrew formulae for tools by [antoninbas](https://github.com/antoninbas).

## Installation

```bash
brew tap antoninbas/tap
```

## Available Formulae

### knotes

A local-first note and activity log manager with hybrid search.

```bash
brew install antoninbas/tap/knotes
```

See [antoninbas/knotes](https://github.com/antoninbas/knotes) for documentation.
